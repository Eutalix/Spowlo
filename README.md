## 🔮 Features

- Download songs, albums, and playlists from Spotify.
- Powered by the latest versions of [spotDL](https://github.com/spotDL/spotify-downloader/) and `yt-dlp`.
- Search for tracks directly within the app.
- Embed synced lyrics into downloaded songs.
- Modern, user-friendly interface built with Jetpack Compose and Material Design 3.
- Dynamic color theming (Material You).
- Written purely in Kotlin, following modern Android development practices.

## ⬇️ Download

For most devices, it is recommended to install the **arm64-v8a** version of the APK.

- Download the latest stable version from the [**GitHub Releases**](https://github.com/BobbyESP/Spowlo/releases/latest) page.

## ⚠️ Warning
Spowlo uses YT Music and YouTube as its audio source. This is because bypassing Spotify's DRM can lead to account bans and legal issues. If YT Music is not available in your country, you can still use YouTube as an audio provider or use a VPN.

## 💻 Building from Source

This project is now self-contained and easy to build.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Eutalix/Spowlo.git
    cd Spowlo
    ```
2.  **(Optional but Recommended) Set up Spotify API Credentials:**
    The app is designed to work without any special setup, using an anonymous token generated by `spotdl`. However, for a more stable search experience, you can use your own Spotify API credentials:
    *   Create a file named `local.properties` in the root directory of the project.
    *   Go to the [Spotify Developer Dashboard](https://developer.spotify.com/dashboard/) and create an application to get your `Client ID` and `Client Secret`.
    *   Add your credentials to `local.properties`:
        ```properties
        CLIENT_ID="YOUR_SPOTIFY_CLIENT_ID_HERE"
        CLIENT_SECRET="YOUR_SPOTIFY_CLIENT_SECRET_HERE"
        ```
3.  **Build the project** using Android Studio or the Gradle wrapper:
    ```bash
    ./gradlew assembleRelease
    ```
